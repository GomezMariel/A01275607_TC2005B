Actividad: CSS Cheat Sheet

1. Border

Como su nombre lo indica, coloca un borde a un elemento. Tiene varias propiedades como el color, y estilo

2. Border-left, border-right, border-bottom, border-top

Como el elemento descrito anteriormente, crea borde en un elemento; sin embargo, solo se modifica la parte referente a la palabra.

3. Border radius

Border radius hace que las esquinas de nuestros bordes sean redondas. Al cambiar los valores, se pueden cambiar las formas del border.

Los bordes de este
ejemplo están redondeados
con 35% y 10%.

4. Background-image

Como su nombre lo dice, es poder ponerle una imagen de fondo a un elemento. La primera capa que se declara, es la más cercana al usuario.

Para este ejemplo usé una foto para poder poner
el background imagen. Sin embargo, también se
podría usar un gradiente para ponerlo de fondo de este elemento

5. Color

Cambia de color el texto. Se pueden usar palabras predefinidas para seleccionar el color, o también por medio de rgb, hsl, hwb, lab, entre otros.

Para ejemplificar, cambiamos de color el texto de este párrafo utilizando id para seleccionarlo y rgb para elegir el color.

6. Font

Establece las propiedades de la fuente que queremos utilizar, y también establece la fuente de un elemento.

Para este ejemplo cambiamos la font de este párrafo usando tamaño 14, tipografía Times New Roman y haciéndola bold.

7. Font-family

Font-family nos da una lista de fuentes que podemos utilizar. La lista está separa por comas y está en orden de prioridad. Ponemos varias fuentes, porque no sabemos si la que seleccionamos se podrá usar siempre.

Usando font family, cambiamos este texto a la familia de Trebuchet MS.

8. Font-style, font weight

Font style nos indica que estilo tendrá las fuentes, si es normal, oblique o italic. Font weight indica que tan bold va a estar un elemento. Puede depender de la font family que se esté usando.

Para ejemplificar font style y wieght, este texto es oblique 45deg y weight 900.

9. Width, height

Usadas para modificar el alto y ancho de un elemento. Esto no incluye bordes o padding.

11. Margin

Es la propiedad que puede modificar el margen, tomando en cuenta los cuatro lados del elemento. Se puede usar margin left, right, top y bottom.

Este ejemplo tiene un margen de 10px 80px 10px 80px.

12. Padding

Pone un colchón o relleno a los cuatro lados del elemento. Nos permite editar sus propiedades. Al igual que en otros elementos, se puede editar cada lado usando let, right, bottom y top.

Este elemento tiene un padding de 10px 30px 20px 30px.

13. Overflow

Si el contenido de un elemento es muy grande para su contenedor, se puede establecer el comportamiento que tendrá el contenido.

Para ver el overflow, definimos un width, y height para este contenido, y al ser muy pequeño no cabe todo el texto aquí, por lo que podemos ver como funciona el overflow scroll.

14. Display

Display establece la manera en la que se ven los elementos. Esto también tiene la capacidad de afectar a sus hijos.

Este elemento está con un display de inline-block.

15.Position

Nos da la oportunidad de cambiar la forma en la que un elemento se posiciona en la página.

Para este ejemplo no quería que elemento modificar tanto la página, por lo que un position sticky no era una opción. Por lo tanto, usé un position relative para que ahí se quedara.

16. Filter

Filter se aplica principalmente en elementos gráficos como imágenes, backgrounds o bordes. Contiene varias funciones que generen un efecto sobre la imagen. En el ejemplo, la imagen tiene un efecto blur de 10px.
