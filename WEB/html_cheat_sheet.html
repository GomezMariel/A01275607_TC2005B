Actividad: HTML Cheat Sheet

1. a - Hyperlink

El anchor sirve para crear un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma páginas

2. p - Paragraph

Cómo su nombre lo indica, crea un párrafo, el cual es representado com un bloque de texto, separdo por líneas en blanco entre bloques o first-line indentation. Los párrafos son block-elements.

3. h1, h2, h3 ... - Headings

Section Headings se usan desde h1 (el elemnto más importante), hasta h6 (el nivel más bajo de heading). No se deben usar para redimensionar el texto, esto se hace con otras funciones. h1 se puede usar varias bveces solo si estos no estñan anidados, sino es considerado mala práctica.
