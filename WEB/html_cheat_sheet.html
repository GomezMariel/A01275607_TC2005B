Actividad: HTML Cheat Sheet

1. a - Hyperlink

El anchor sirve para crear un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma páginas.

Link a una página que lo explica mejor

2. p - Paragraph

Cómo su nombre lo indica, crea un párrafo, el cual es representado cómo un bloque de texto, separado por líneas en blanco entre bloques o first-line indentation. Los párrafos son block-elements.

Esta línea de texto está escrita en párrafo.

3. h1, h2, h3 ... - Headings

Section Headings se usan desde h1 (el elemento más importante), hasta h6 (el nivel más bajo de heading). No se deben utilizar para redimensionar el texto, esto se hace con otras funciones. h1 se puede usar varias veces solo si estos no están anidados, si no es considerado mala práctica.

Este texto está mostrado como un heading h5

4. div - Content Division

Div es un contenedor puro de flujo. Es usado para dar formato marcando secciones. Es un block level element.

Este es un ejemplo de como en esta sección se puede incluir más texto.

5. span - Span

Span es un contenedor genérico. Se usa para agrupar elementos y darles el mismo estilo. Es similar a div, con la diferencia de que este es un elemento inline.

Como ejemplo, utilizamos span con verde.

6. strong - Strong Importance

Strong es un elemento que indica urgencia, seriedad o gran importancia. Al compilarlo se ve como bold type.

Este ejemplo es para ver como esto es más importante.

7. i - i

Sirve para darle una voz alternativa al texto. Se usa para resaltar un término técnico, frases en otro idioma, nombres, entre otros. Al reproducirlo se ve como itálicas.

Este ejemplo es para ver como este ejemplo es parte de la tarea html cheat sheet

8. u - Unarticulated Annotation (Underline)

Underline es un elemento inline que indica que tiene una anotación no textual. Al reproducirlo se pone un subrayado y se puede alterar usando CSS.

Este ejemplo es para ver como usamos underline palbas mal escritas.

9. b - Bring Attention To

Es usado para que el lector le ponga atención a lo que se desea resaltar. Se usa principalmente para resaltar keywords o nombres importantes. Al reproducirlo, aparece en bold.

Este ejemplo es para resaltar una palabra con b.

10. strike - Strike

Strike es un elemento que ha sido removido de varios web standards. Coloca una línea horizontal en el texto.

Usamos s en lugar de strike para poner la línea horizontal sobre la palabra.

11. small - Small

Se usa para colocar side-comments o small prints. Al reproducirlo, se muestra como un tamaño de fuente más pequeño por default.

Escribimos el ejemplo y en letra más pequeña empleamos small.

12. em - Em

Utilizado para enfatizar el texto, usando verbal stress. Se puede visualizar con itálicas.

El ejemplo muestra como debemos usar em ¡ahora!.

13. header - Header

Se usa para representar contenido introductorio de una página, que ayudan en la navegación. Si se describe como banner, puede contener el logo, nombre la compañía, entre otros elementos. Si se describe como section normalmente contiene un heading.

14. section - Section

Cómo su nombre lo indica, sirve para indicar una sección independiente. Para una buena práctica, siempre se debe añadir un heading. Solo se debe usar si no hay un elemento más específico que describa el contenido.

22. form - Form

Form contiene controles interactivos, que se usa principalmente para obtener información del usuario.

24. ul - Under List

ul es una lista de elementos desordenados. No tiene orden numérico

  • Elemento x
    • Primer sub elemento del elemento x
  • Elemento a
  • Elemento d
    • Primer sub elemento del elemento d
    • Segundo sub elemento del elemento d

25. ol - Ordered List

A diferencia de ul, ol si es una lista ordenada, ya sea por orden numérico o alfabético.

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
